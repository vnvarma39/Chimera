How to Run the Model
Open three terminals inside the chimera folder and run:
# Terminal 1 — start the honeypot
python honeypot.py

# Terminal 2 — start the dashboard
streamlit run dashboard.py

# Terminal 3 — connect as attacker (demo mode)
ssh admin@localhost -p 2222
Type any password when prompted. Then run the demo sequence below to generate preliminary results.

3.4  What the Dashboard Shows
The live dashboard at localhost:8501 displays the following panels simultaneously:
•	Command stream — every command in green monospace with MITRE tags firing below each
•	MITRE ATT&CK heatmap — technique counts with progress bars, updates every 2 seconds
•	Attacker profile — auto-classified (Script Kiddie / Intermediate / APT) with threat score 0-100
•	Prompt evolution — shows each evolution event: focus detected, bait files injected
•	Dialogue engine — shows attacker messages and sysadmin replies in real time
•	Canary alerts — red alert fires the moment a planted credential file is accessed
•	Session counter — active sessions, total commands, MITRE tags, canary hits across all sessions
3.5  What Happens on Exit
When the attacker types exit, the system automatically:
1.	Stores a semantic fingerprint of the command sequence in attacker_memory.json
2.	Generates a threat intelligence report as an HTML file in data/reports/
3.	Saves the complete session to sessions.jsonl for dataset export
4.	Logs any canary token accesses for external monitoring
